# Vim Terminal Options

We can open Vim using different options:


## Use a specific vimrc:

```
$ vim -u <vimrc>
```

## Don't use a vimrc:
```
$ vim -u NONE
```

## Display verbose log messages
```
$ vim -V
```

----
## Send Vim to background
```
CTRL+z
```

## Send Vim to the foreground
```
# in the command line
$ fg
```
